c9s

Terminal dashboard for Claude Code. Like k9s, but for your AI sessions.

Homebrew brew install stefanoguerrini/tap/c9s
Go go install github.com/stefanoguerrini/c9s@latest
Then c9s

What it does

  • Zero setup -- reads directly from ~/.claude/
  • All sessions in one view -- across every project
  • Live status -- processing, waiting, done
  • tmux integration -- open, resume, switch sessions
  • Token tracking -- see usage per session
  • Keep alive -- sessions survive when you quit
  • Git worktrees -- see and switch between worktrees
  • Fully configurable -- colors, keys, refresh rate

How it works

c9s reads Claude Code's local files (~/.claude/) to discover all your sessions. No API calls, no network, no daemon.

Each session runs in its own tmux window. Navigate with j/k, open with Enter, create with n. That's it.
