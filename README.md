## 📋 What is c9s?

c9s is a tool that helps you keep track of your terminal sessions. It shows all your running tasks in one place. You can switch between different tasks quickly without losing your work. It works right inside your command prompt and needs no setup.

If you use Claude Code or many terminal sessions at once, c9s keeps your work tidy and easy to follow.
